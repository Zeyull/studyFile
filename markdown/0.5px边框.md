# 0.5px边框	

## 为什么会要求0.5px边框？

​	因为0.5px的边框在不同浏览器的效果是不同的，有些浏览器会自动转换成1px的宽度。所以我们需要写能兼容的方法。

## 单边框

### 方法一：border + border-image + linear-gradient

```html
/*这种方法出来的效果会让原本的200px变成200.5px，整体就是201px*/
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .border{
            height: 200px;
            width: 200px;
            margin: 0 auto;
            background-color: red;
            border-bottom: 1px solid transparent;
            border-image:linear-gradient(to bottom,transparent 50%,green 50%);
            /*图片边框向内偏移，不懂其原理*/
            border-image-slice: 0 0 100% 0;	
        }
    </style>
</head>
<body>
    <div class="border"></div>
</body>
<script>
</script>
</html>
```



### 方法二：伪元素 + background-image + linear-gradient

```html
/*这种方法出来的效果会导致200px变成199.5px*/
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .border{
            height: 200px;
            width: 200px;
            margin: 0 auto;
            background-color: red;
            position: relative;
        }

        .border::before{
            content: " ";
            position: absolute;
            bottom: 0;
            left: 0;
            width: 100%;
            height: 1px;
            background-image:linear-gradient(to bottom,transparent 50%,blue 50%);
        }
    </style>
</head>
<body>
    <div class="border"></div>
</body>
<script>
</script>
</html>
```



### 方法三：定位 + 伪元素 + transform缩放（scale）

```html
/*这种方法出来的效果会导致200px变成199.5px*/
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .border{
            height: 200px;
            width: 200px;
            margin: 0 auto;
            background-color: red;
            position: relative;
        }

        .border::before{
            content: "";
            position: absolute;
            bottom: 0;
            left: 0;
            width: 100%;
            height: 1px;
            background-color: green;
            transform-origin: left bottom;
            transform: scaleY(0.5);
        }
    </style>
</head>
<body>
    <div class="border"></div>
</body>
<script>
</script>
</html>
```



## 多边框

### 方法一：定位 + 伪元素 + transform缩放(scale)

```html
/*对元素200px的宽高没有影响*/
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .border{
            height: 200px;
            width: 200px;
            margin: 0 auto;
            background-color: red;
            position: relative;
        }

        .border::before{
            content: " ";
            position: absolute;
            width: 200%;
            height: 200%;
            border: 1px solid blue;
            transform-origin: -1px -1px;
            transform: scale(0.5);
        }
    </style>
</head>
<body>
    <div class="border"></div>
</body>
<script>
</script>
</html>
```
